## User Info Form

### About

This is a mini project to experiment with reading form data. I'm practicing this by reading the input and adding the user's info to a list underneath.

### Versions

Currently, only version 1 is complete. It reads the input and adds it to the list. The plan for version 2 is to add some styling and a user id. Version 3 will have a validation (checking for empty fields) and local storage. Version 4 will have a delete button for each user.
